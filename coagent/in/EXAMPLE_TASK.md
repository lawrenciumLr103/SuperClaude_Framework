# Example Task

This is an example of how to structure a task file for coagent workflow optimization.

## Task Description

Build a REST API with JWT authentication for a user management system.

## Requirements

- User registration and login endpoints
- JWT token generation and validation
- Password hashing with bcrypt
- Role-based access control (admin, user)
- Rate limiting on auth endpoints

## Technical Context

- Framework: FastAPI (Python)
- Database: PostgreSQL
- Existing codebase: None (greenfield)

## Constraints

- Must follow OWASP security guidelines
- API should be documented with OpenAPI/Swagger
- Include comprehensive test coverage
